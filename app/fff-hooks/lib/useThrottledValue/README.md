# What is it

Limits the rate at which a function can be called.

# Usage

When you want to reduce the number of times a function is called, even if the function is triggered frequently.

- Preventing Overload
- Rate Limiting
- Conserving Resources

# Use Cases

- Scroll Events
- Mouse Move Events
- Button Clicks
